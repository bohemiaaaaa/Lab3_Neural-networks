# Лабораторная работа 3: Разработка и исследование свёрточных нейронных сетей для задач классификации изображений.

## Цель работы
Изучить архитектуры свёрточных нейронных сетей (CNN), освоить методы аугментации данных, регуляризации и Transfer Learning для повышения точности классификации изображений на примере датасетов рукописных цифр, пассажиров автобуса и автомобилей.

## Задачи работы
1. Реализовать и обучить свёрточную нейронную сеть на датасете MNIST (PyTorch) с соблюдением заданных параметров.
2. Реализовать и обучить свёрточную нейронную сеть на датасете "Пассажиры автобуса" (TensorFlow) с учётом дисбаланса классов.
3. Реализовать классификацию автомобилей с использованием предобученной модели MobileNetV2 (Transfer Learning) и метода постепенной разморозки слоёв (fine-tuning)
